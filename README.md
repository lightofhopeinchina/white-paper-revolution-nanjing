# 南京白纸革命活动召集帖
近年来，习近平大兴土木、发动狂魔式基建，疯狂榨取百姓劳动成果，青年失业率飙升、生育率下降、国家消费活力降低，百姓生活苦不堪言，这一切都与习近平的独裁统治带来的畸形的权力结构脱不了干系。为了对这种高压独裁统治进行抗议，南京人民将与各地区统一战线，在成都大运会期间（7.28~8.8）举行“快闪”抗议活动。“快闪”活动的基本安排如下：

1. 本活动无组织者，属于原子化的抗议，欢迎各种反习近平的团队参加抗议活动，即使你是党员我们也欢迎，我们知道很多中共党员也是反习近平的独裁，反习近平独裁是我们最大的公约数。

2. 快闪现场携带白纸，多带几张，如果被询问，就说有用，理由自己编，别全部一样，也可以遗弃一些白纸供其他人使用。 （另：家里有存货的可以带几本习近平的著作，或者是一些经典名著之类的，避免被查，活动开始时将空白的扉页撕下来当白纸用）

3. 上街进行抗议，口号可以任选，在现场达成一致，可供参考的口号有：“共产党下台”、“习近平下台”、“罢免独裁国贼习近平”、“It's our duty”、“不自由，毋宁死”。

4. 抗议过程中尽量往人多的地方前进，靠近现场的可以多拍视频，知晓的人越多，快闪活动的影响越大。

为了所有抗议人员的安全，有一下几点须知：

1. 不要透露任何个人信息，无论是线上还是线下。

2. 不要联系其他人共同参加抗议，即使商量一起行动，也不要告诉对方具体的时间和地点。

3. 在参加快闪活动前，清理掉电脑、手机里面的敏感内容，防止抗议过程中或者事后搜查。

4. 抗议过程中可以佩戴口罩、鸭舌帽等，避免事后通过影像资料清算。
